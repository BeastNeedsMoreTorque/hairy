[![Package version][2]][3]
[![Build status][4]][5]

A RESTful CRUD web app.

## Installation

Add it to your Cabal file:

```
library
  build-depends:
    hairy ==0.1.*
```

Or install it manually:

``` sh
$ cabal update
$ cabal install hairy-0.1.1
```
